よく使うコマンドは覚えてるんだけど、たまに使うコマンドは忘れちゃって毎回ググってる気がするのでメモ。

バージョン確認

```coffeescript
npm -v
```

npm自身のアップデート

```coffeescript
npm i -g npm
```

パッケージをアインインストール

```sql
npm uninstall <package>
```

パッケージの更新情報を確認

```coffeescript
npm outdated
```

dependenciesだけインストール

```stylus
npm i --production
```
