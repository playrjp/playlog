よく使うコマンドは覚えてるんだけど、たまに使うコマンドは忘れちゃって毎回ググってる気がするのでメモ。

バージョン確認

```
npm -v
```

npm自身のアップデート

```
npm i -g npm
```

パッケージをアインインストール

```
npm uninstall <package>
```

パッケージの更新情報を確認

```
npm outdated
```

dependenciesだけインストール

```
npm i --production
```
