メジャーリリースされたとのことで、NPMをv3へアップデート。

```bash
$ npm i -g npm
```

とりあえずモジュールのインストールを再実行してみた。

フラット化の結果25のモジュールインストールに対して300個がnode\_modules内に召還された。  
時間もなんか長かったな…。この辺は徐々に高速化されていくのだろうか。  
間違ってもpackege.json管轄外でインスコしないように気を付けなければ、不幸な孤児が生まれそうである。
